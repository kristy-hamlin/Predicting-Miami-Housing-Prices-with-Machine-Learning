Miami Housing Price Prediction using Decision Tree Regressor
Kristy Hamlin
Machine Learning

This repository contains the following:
1. A Jupyter Notebook showing the data exploration, preprocessing, and creation of an ML model to predict Miami Housing Prices.
2. The original dataset used from Kaggle.
3. A PDF report describing the project and analysis.
4. A PDF of a PowerPoint Presentation summary of the project.
